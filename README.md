SpiceHub Restaurant Website UI Redesign

This project is a complete UI redesign of a restaurant website focused on improving usability, visual appeal, and responsiveness.

Deliverables Included:

* Existing website redesign concept
* Modern high-fidelity desktop homepage
* Mobile responsive homepage
* Improved navigation, CTA hierarchy, and visual consistency
* Restaurant-focused branding and layout

Tools Used:

* Figma
